<h1>Docker - Setups</h1>

# EJECUTAR CONTENEDOR

## Ejecutar contenedor con el siguiente comando

```bash
  docker-compose up -d
```

### Ingresar a la siguiente ruta para ver el proyecto

```bash
  http://localhost:8081/
```

### Elegir el lenguaje del phpmyadmin, y ingresar el usuario y la contraseña

```bash
  Usuario: user
  Contraseña: 1234
```
